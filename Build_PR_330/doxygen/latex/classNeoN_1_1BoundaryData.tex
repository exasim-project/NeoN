\subsubsection*{template$<$typename T$>$\newline
class Neo\+N\+::\+Boundary\+Data$<$ T $>$}
Represents the boundary fields for a computational domain. 

The \doxylink{classNeoN_1_1BoundaryData}{Boundary\+Data} class stores the boundary conditions and related information for a computational domain. It provides access to the computed values, reference values, value fractions, reference gradients, boundary types, offsets, and the number of boundaries and boundary faces.
